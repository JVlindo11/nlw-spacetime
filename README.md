## 🖥️ projeto 
Esse é um projeto web Responsivo de uma capsula do tempo para exibir memorias em uma linha do tempo.

## 🚀 Tecnologias
Esse projeto foi desenvolvido durante a NLW da Rocketseat com as seguintes tecnologias:

- HTML
- CSS
Git e Github
